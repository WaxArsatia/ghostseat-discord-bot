# Suggested Commands

## Run
```bash
bun run index.ts
```

## Install dependencies
```bash
bun add discord.js @discordjs/voice
bun add -d @types/bun typescript
```

## Type check
```bash
bun tsc --noEmit
```

## Register slash commands (deploy script)
```bash
bun run src/scripts/deploy-commands.ts
```
